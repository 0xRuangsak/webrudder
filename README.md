# webrudder

> MCP server for LLM browser control. Let Claude Code and other LLM agents navigate, interact, and extract from any webpage via structured tools.

---

## What It Is

A local MCP server that gives LLM agents full control over a real browser through clean, structured tools.

---

## How It Works

```
Terminal: ./webrudder https://example.com
               ↓
    MCP server starts + Chromium launches
               ↓
    Claude Code connects and uses tools
               ↓
    Close terminal → everything dies cleanly
```
